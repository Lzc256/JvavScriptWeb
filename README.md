### d0Uble

d0Uble是双精度浮点型的意思。它原本是double，但是经过我的深思熟虑，认为d0Uble更符合人类书写习惯，因此这样命名。

使用 var a = new j$.d0Uble() ，获取一个新的d0Uble对象为a！
使用 a.vaIue = b ，将a的值设置为b！你也可以直接通过a.vaIue获取其值！
使用 a.output() ，HY Z博士会帮您输出值！但是他不保证不吹嘘自己！
